# the-app

This is the code for the [sbrdbry.com](https://sbrdbry.com) website. Run it locally:

```bash
$ git clone https://github.com/sbrdbry/the-app.git
$ cd the-app
$ npm i
$ docker compose up -d
```

Once docker is up you can visit http://localhost:8090 in the browser.
